<h1>Веб-приложение для подбора фильмов</h1>

Это простое веб-приложение на базе Django, которое помогает пользователю подобрать фильм, соответствующий его настроению и предпочтениям. Система не просто ищет по категориям, а использует алгоритмы машинного обучения для поиска «соседей» по набору признаков.

---
<h3>Как это работает:</h3>
<p>
Данные из базы Django передаются в DataFrame, где проходят через алгоритм KNN и выдает 3 ближайших соседей.
</p>

---
<h3>Как обрабатываются признаки и как KNN находит соседей.</h3>
Из за большого разброса признака Year и остальных, признаки нормализуются с помощью MinMaxScaler(с расчетом на то что в данных нет выбросов)

Алгоритм NearestNeighbors запоминает всю выборку и считает **евклидово расстояние** до каждой точки, и выдает те K точек, у которых расстояние наименьшее.

<h3>Описание признаков:</h3>

- Happy (1-10) — уровень позитива.

- Action (1-10) — Экшен.

- Weird (1-10) — необычность сюжета.

- Year — год выпуска.

- Rating — рейтинг фильма на IMDB.

---
<h3>Как запустить проект:</h3>

1) Клонирование репозитория
```python
git clone https://github.com/Pudgita/MovieRecomendation
cd MovieRecomendation
```
2) Установка необходимых библиотек
```python
pip install -r requirements.txt
```
3) Применение миграции
```python
python manage.py migrate
```

4) Запуск сервера
```python
python manage.py runserver
```
